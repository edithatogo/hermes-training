# Specification: Qwen3 V4 Dataset Publication Scope

Prepare a concrete dataset publication scope package for the Qwen3 v4 targeted
adapter without publishing any data.

Acceptance criteria:

- Record the current dataset composition, provenance, audit status, and
  redistribution caveats.
- Separate adapter-release evidence from public dataset-release approval.
- Define a recommended safe publication scope and explicit exclusions.
- Add a dataset-card draft for the current v4 materialized dataset.
- Keep public dataset publication blocked pending human scope approval.
- Validation passes.
