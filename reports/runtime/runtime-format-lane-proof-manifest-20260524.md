# Runtime Format Lane Proof Manifest

Date: 2026-05-24

This manifest turns `RUNTIME_FORMAT_LANES.yaml` into executable proof-card work. It does not claim any new runtime proof by itself.

The machine-readable queue is:

```text

```

## Artifact Root

Proof cards generated by `scripts/create_runtime_format_lane_card.py` default to:

```text

```

## Generator

```bash
source scripts/env.sh
./.venv/bin/python scripts/create_runtime_format_lane_card.py \
  --lane-id mlx-native \
  --model Qwen/Qwen3-4B-MLX-4bit \
  --endpoint http://127.0.0.1:8080/v1 \
  --command 'mlx_lm.server --model Qwen/Qwen3-4B-MLX-4bit --port 8080'
```

Use `--print` to preview without writing a file.

Generate or validate the full first queue:

```bash
source scripts/env.sh
./.venv/bin/python scripts/create_runtime_format_proof_queue.py --check
./.venv/bin/python scripts/create_runtime_format_proof_queue.py
```

## Promotion Rule

A lane proof card can promote a model only to the evidence type it actually proves:

- runtime smoke proves local serving or invocation only
- training run proves adapter production only
- teacher API output proves teacher/evaluator evidence only
- publication claims require benchmark and license gates in addition to runtime or training proof
